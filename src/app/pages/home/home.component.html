Funciones de los Sensores
  • Agua: Este sensor mide la cantidad de agua disponible para la planta.
  • Temperatura: Este sensor mide la temperatura del ambiente en el que se encuentra la planta.
  • Humedad: Este sensor mide la humedad del ambiente en el que se encuentra la planta.
  • Lluvia: Este sensor detecta si está lloviendo o no.
  • Suelo: Este sensor mide la calidad del suelo en el que se encuentra la planta.
  • Luz: Este sensor mide la cantidad de luz que recibe la planta.
  • Movimiento: Este sensor detecta si hay algún movimiento alrededor de la planta.
  • Vibración: Este sensor detecta si hay alguna vibración en el suelo donde se encuentra la planta.
